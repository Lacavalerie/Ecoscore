LANCEMENT LOCAL
───────────────
  Mac    → double-cliquer sur LANCER.command
  Windows→ double-cliquer sur LANCER.bat
  Terminal: node server.js

  Puis ouvrir : http://localhost:3000
  Mot de passe : ecoscore2025

DÉPLOIEMENT RENDER (URL publique)
──────────────────────────────────
  1. Mettre les fichiers sur GitHub
  2. render.com → New Web Service → GitHub repo
  3. Build Command : (laisser vide)
  4. Start Command : node server.js
  5. Variables d'environnement :
     ACCESS_PASSWORD = votre_mot_de_passe
     ANTHROPIC_API_KEY = sk-ant-...
     PORT = 10000

  ⚠️  Sur Render : le fichier config.json est éphémère.
  Mettez votre clé API dans la variable ANTHROPIC_API_KEY.
