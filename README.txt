LANCEMENT
─────────
  Mac    → double-cliquer sur LANCER.command
  Windows→ double-cliquer sur LANCER.bat
  Terminal: node server.js

Ouvrir dans le navigateur : http://localhost:3000

MODULES
───────
  ▦ Tableau de bord    — Vue globale ESG + alertes
  ◉ Salariés           — Gestion des collaborateurs
  ◈ Fiches de poste    — Descriptions de postes
  ◎ Fournisseurs       — Panel & scores ESG
  ◇ Données ESG        — Saisie E/S/G
  ◻ Documents          — Bibliothèque documents
  ⚖ Législation        — 24 textes FR & UE référencés
  📜 Politiques RSE     — Éditeur + Assistant IA Claude
  ≡ Rapport CSRD       — Génération rapport
  ⊙ Paramètres         — Profil + clé API Anthropic

ASSISTANT IA RSE
────────────────
  Pour activer l'assistant IA dans le module RSE :
  1. Allez dans Paramètres (bas de la barre de gauche)
  2. Entrez votre clé API Anthropic (console.anthropic.com)
  3. Enregistrez
  L'assistant utilise Claude pour rédiger et analyser
  vos politiques RSE.

PRÉREQUIS
─────────
  Node.js (nodejs.org, version LTS)
